\# AthleteOS



AthleteOS es una plataforma de monitorización de carga y soporte a la toma de decisiones para equipos de fútbol.



\## Funcionalidades



\- Dashboard ejecutivo

\- Dashboard individual

\- ACWR y EWMA

\- Action Center

\- Heatmap de riesgo

\- Importación de datos GPS/RPE



Desarrollado con Dash + Python.
